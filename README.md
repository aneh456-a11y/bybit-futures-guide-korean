## 1. 상단바 & 종목 헤더

### Search Coin / (코인 검색)
- **위치**: 상단바 왼쪽, 검색 입력창
- **용도**: 다른 코인으로 거래 화면 바꿀 때
- **동작**: 종목 입력 → 선택 → 화면 전환
- **예시**: `ETHUSDT` 입력 → 이더리움 선물로 이동

### BTCUSDT / (BTC테더 선물)
- **위치**: 종목 헤더, 큰 심볼 표시
- **용도**: 현재 보는 계약 확인
- **동작**: 클릭 시 종목 선택 UI 열림

### Last Price / (현재가)
- **위치**: 큰 숫자로 표시 (ex: 69,901.50)
- **용도**: 가장 최근 체결 가격 확인
- **동작**: 실시간 변경
- **예시**: 69,900 표시 → "방금 69,900에 거래됨"

### Index Price / (지수 가격)
- **위치**: Index Price 69,931.58
- **용도**: 여러 현물 가격 종합 기준값 (강제청산 기준)
- **동작**: 실시간 변경

### Change % / (등락률)
- **위치**: Change % -494.10 (-0.70%)
- **용도**: 24시간 대비 변동률
- **예시**: -0.70% → "하루 전 대비 0.7% 하락"

### Funding Rate / Countdown / (펀딩비)
- **위치**: Funding Rate 0.0001% / 06:31:34
- **용도**: 무기한 선물 포지션 보유 비용/수익 확인
- **동작**: 시간은 감소, 비율은 주기적 갱신
- **예시**: 0.01% 포지션이면 정산 시 비용/수익 발생 가능

## 2. 차트 영역

### Chart / (차트)
- **용도**: 가격 캠들과 지표 표시
- **동작**: 차트 화면 표시

### Time Interval (1s, 1m, 5m, 1h, 1d...)
- **용도**: 캠들 1개가 나타내는 시간 단위 변경
- **예시**: 1h 선택 → 1캠들 = 1시간

### Standard / TradingView / Depth
- **비교**: Standard=기본, TradingView=그리기도구 강함, Depth=호가 누적 시각화

### Recent Trades / (최근 체결)

## 4. 주문창 & 계정

### Margin Mode: Isolated / (격리)
- **용도**: 포지션별 증거금 분리 (Cross=공유)
- **예시**: 격리에 100 USDT 배정 → 그 범위내서만 리스크

### Leverage 10.00x / (레버리지)
- **용도**: 같은 포지션에 필요한 증거금 조절
- **예시**: 10x에 1,000 USDT 포지션 = 약 100 USDT 증거금

### Order Type: Limit / Market / Conditional
- **Limit**: 지정가격으로 대기 주문
- **Market**: 즉시 체결 시도
- **Conditional**: 특정 조건 만족 시 주문 발동

### Price / Quantity / Value / Cost
- **Price**: 지정가 입력
- **Quantity**: 수량 입력 (BTC)
- **Value**: 주문 규모 (USDT)
- **Cost**: 필요 증거금

### TP/SL / Post-Only / Reduce-Only / GTC
- **TP/SL**: 익절/손절 자동 설정
- **Post-Only**: 즉시체결 방지 (메이커만)
- **Reduce-Only**: 포지션 감축만 허용
- **GTC**: 취소 전까지 주문 유지

### Long / Short 버튼
- **Long**: 상승 방향 포지션 열기
- **Short**: 하락 방향 포지션 열기

### Calculator / (계산기)
- **용도**: 청산가/수익 등 계산 도구

### Request Demo Funds
- **용도**: 데모 자금 충전

### Contract Details
- **표시**: Mark Price, Risk Limit, Contract Value 등
- **용도**: 계약 상세 정보 확인

## 5. 하단 탭들

### Open Orders / (미체결 주문)
- **용도**: 체결 안 된 주문 확인/취소

### Positions / (포지션)
- **용도**: 현재 열려 있는 포지션 확인
- **표시**: 진입가, 수량, 손익, 청산가 등

### Order History / (주문 내역)
- **용도**: 과거 주문 기록 조회

### Trade History / (거래 내역)
- **용도**: 실제 체결 기록 조회

### Assets / (자산)
- **용도**: 잔고, 사용가능 금액 확인

## 추가 팁

### 용어 설명
- **롤/숙**: Long=매수(상승 기대), Short=매도(하락 기대)
- **레버리지**: 증거금 대비 포지션 배수 (10x = 10배)
- **마크 가격**: 평가/청산 기준가
- **편딩비**: 무기한 선물에서 주기적 발생하는 보유 비용/수익

### 기본 흐름
1. 종목 선택 (Search Coin)
2. 레버리지/마진모드 설정
3. 주문 방식 선택 (Limit/Market)
4. 가격/수량 입력
5. Long/Short 클릭
6. Positions 탭에서 포지션 확인
7. 종료 시 청산 주문
