# TradingView 도구 가이드

> TradingView 차트에서 사용하는 주요 도구들을 초보자 기준으로 설명합니다.  
> 각 도구의 뜻, 용도, 사용법, 차이점, 실용성을 포함합니다.

---

<details open>
  <summary><b>▶ Cross (십자선 / 커서)</b></summary>

**뜻**  
차트에서 정확한 가격과 시간을 확인할 수 있는 십자 모양 커서

**용도**  
- 특정 시점의 정확한 가격 확인  
- 두 지점 간의 가격 차이, 시간 간격, 변동률(%) 측정  
- 패턴 분석 시 정확한 위치 파악  

**사용법**  
1) 왼쪽 툴바에서 Cross 아이콘 클릭  
2) 차트에 마우스 갖다대면 십자선 나타남  
3) 클릭하면 그 시점의 OHLC 데이터 표시  

**초보자 팁**  
기본으로 선택해두고 차트 보는 게 제일 편함. 다른 도구 쓰다가도 ESC 누르면 Cross로 돌아옴.

**차이점**  
- Cross: 클릭해도 차트에 아무것도 안 그려짐 (확인만)  
- Dot: 차트에 점이 남음 (표시)  
- Text: 글자를 남김 (설명)  

**실용성**: ★★★★★

</details>

<details>
  <summary><b>▶ Trend Line (추세선)</b></summary>

**뜻**  
고점과 고점, 저점과 저점을 연결해서 추세의 방향과 강도를 시각화하는 선

**용도**  
- 상승 추세선: 저점 연결 → 지지선 역할  
- 하락 추세선: 고점 연결 → 저항선 역할  
- 추세선 돌파 = 추세 전환 신호  

**사용법**  
1) Trend Line 클릭 (Alt + T)  
2) 시작점 클릭 → 끝점 클릭  
3) 상승장: 2개 이상의 저점 연결  
4) 하락장: 2개 이상의 고점 연결  

**초보자 팁**  
- 3점 터치 원칙: 추세선이 3번 이상 터치되면 신뢰도 높음  
- 추세선 + 거래량 증가 돌파 = 강한 신호  

**차이점**  
- Trend Line: 끝점 이후로만 연장  
- Ray: 한쪽 방향으로만 무한 연장  
- Extended Line: 양쪽 무한 연장  

**실용성**: ★★★★★

**예외**  
변동성이 너무 큰 코인(밈코인, 신규 상장)에서는 추세선이 자주 깨짐. 이런 경우 수평선이 더 유용.

**(Trend Line) vs (Gann Fan 1x1) 비교**  
- Trend Line: 자유롭게 그림, 주관적  
- Gann Fan 1x1: 45도 고정, 기하학적  
**결론**: Trend Line이 초보자에게 더 실용적

</details>

<details>
  <summary><b>▶ Dot (점 / 마커)</b></summary>

**뜻**  
차트 위에 점을 찍어서 중요 지점을 표시하는 마킹 도구

**용도**  
- 중요한 가격대나 시점 마킹  
- 지지선/저항선 터치 지점 표시  
- 진입/청산 지점 기록  

**사용법**  
1) Dot 아이콘 클릭  
2) 원하는 위치 클릭하면 점 찍힘  
3) 우클릭 → Settings로 색상/크기 조정  
4) 삭제는 Eraser 또는 Delete 키  

**초보자 팁**  
중요 지점에 빠르게 표시할 때 좋음. 텍스트보다 가볍고 빠름.

**실용성**: ★★★☆☆

</details>

## 추가 정보

### TradingView 단축키
- `ESC`: Cross 모드로 복귀
- `Alt + T`: Trend Line 선택
- `Delete`: 선택한 도구 삭제
- `Ctrl + Z`: 실행 취소
- `Ctrl + Y`: 다시 실행

### 도구 관리 팁
1. 자주 쓰는 도구를 즐겨찾기에 추가
2. 도구 색상/두께를 템플릿으로 저장
3. 차트 정리: 오래된 도구는 주기적으로 삭제
